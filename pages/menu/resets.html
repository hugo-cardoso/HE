Reset's

Android Lollipop 5.0

Desligado
  1. Se o smartphone estiver ligado, desligue-o.
  2. Pressione o botão Diminuir volume e o mantenha pressionado, pressione também o botão Liga/desliga durante 3 segundos, após isso solte os dois.
  3. Pressione o botão Diminuir volume uma vez para destacar a opção "Recovery mode" (Modo de recuperação).
  4. Pressione o botão Aumentar volume para iniciar o modo de recuperação. Você verá a imagem do robô Android com um ponto de exclamação vermelho e escrito "Nenhum comando".
  5. Pressione o botão Liga/desliga e mantenha-o pressionado. Mantendo o botão Liga/desliga, pressione e solte o botão Aumentar volume uma única vez.
  6. Pressione o botão Diminuir volume três vezes para destaca "Wipe data/factory reset" (Limpar dados/configuração original) e pressione o botão Liga/desliga para selecionar essa opção.
  7. Pressione o botão Diminuir volume sete vezes para destacar "Yes -- delete all user data" e pressione o botão Liga/desliga para selecionar essa opção.
  8. Depois que a redefinição for concluída, pressione o botão Liga/desliga para selecionar a opção "reboot system now" (Reiniciar sistema).
Ligado
  1. Acesse: Menu> Configurar> Fazer backup e redefinir> Restaurar padrões de fábrica> Restaurar telefone.

Android Lollipop 5.1

Desligado
  1. Se o smartphone estiver ligado, desligue-o.
  2. Pressione o botão Diminuir volume e o mantenha pressionado, pressione também o botão Liga/desliga junto, até que o smartphone seja ligado. Você verá a palavra "Start" (Iniciar) com setas abaixo dela.
  3. Pressione o botão Diminuir volume duas vezes para destacar a opção "Recovery mode" (Modo de recuperação).
  4. Pressione o botão Liga/desliga para iniciar o modo de recuperação. Você verá a imagem do robô Android com um ponto de exclamação vermelho e escrito "Nenhum comando".
  5. Pressione o botão Liga/desliga e mantenha-o pressionado. Mantendo o botão Liga/desliga, pressione e solte o botão Aumentar volume uma única vez.
  6. Pressione o botão Diminuir volume três vezes para destaca "Wipe data/factory reset" (Limpar dados/configuração original) e pressione o botão Liga/desliga para selecionar essa opção.
  7. Pressione o botão Diminuir volume seis vezes para destacar "Userdata only" e pressione o botão Liga/desliga para selecionar essa opção.
  8. Depois que a redefinição for concluída, pressione o botão Liga/desliga para selecionar a opção "reboot system now" (Reiniciar sistema).
Ligado
  1. Acesse: Menu> Configurar> Fazer backup e redefinir> Restaurar padrões de fábrica> Restaurar telefone.
